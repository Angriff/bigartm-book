Этот ноутбук использовался на воркшопе во время летней школы ISMW FRUCT 2016. 
Ноутбук состоит из двух частей: первая знакомит участников с библиотекой и показывает пример построения тематического навигатора по материалам сайт о популярной науке,
во второй настраивается модель для предсказания пола пользователя социальной сети по опубликованным им/ей материалам.
